# OMR-Virtual-Assistance
AI-Powered Restaurant Chatbot built with n8n  OMR Virtual Assistant is a complete AI restaurant automation agent built using n8n, Google Gemini, and Google Sheets. It is designed to handle restaurant conversations, orders, and FAQs in a simple, scalable, and production-friendly way.  This agent works as a multi-language restaurant .English,Urdu,Hin
